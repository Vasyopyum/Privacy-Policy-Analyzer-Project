# Лабораторная работа 1: Текстовый анализатор

## Бизнес-задача

Необходимо реализовать модуль для базового анализа текста. Программа должна принимать на вход строку и возвращать словарь со статистикой.

## Требования к реализации

В файле `analyzer.py` находится каркас функции `analyze_text`. Вам необходимо реализовать её логику так, чтобы она возвращала словарь со следующими ключами:

| Ключ | Тип | Описание |
|------|-----|----------|
| `words_count` | `int` | Общее количество слов |
| `unique_words` | `int` | Количество уникальных слов (без учёта регистра) |
| `longest_word` | `str` | Самое длинное слово |

### Ограничения и краевые случаи (Edge Cases)

1. Знаки препинания **не должны** учитываться как часть слова (слова `"Привет,"` и `"привет"` — это одно и то же слово).
2. Если на вход передана пустая строка, функция должна вернуть словарь с нулевыми/пустыми значениями:
   ```python
   {"words_count": 0, "unique_words": 0, "longest_word": ""}
   ```
